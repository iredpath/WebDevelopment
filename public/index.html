Form Maker

A basic user interface for creating, updating, and answering forms. Developers can create forms, and view or download responses as CSV files. Submitters can fill out and submit all of these forms. This application uses the MEAN stack, consisting of MongoDB, Express, Angularjs, and NodeJS.

Virtual Movie Library

A simple, elegant way to display your movie collection. Create lists to show off what you own, what you wish you had, or any other grouping you can think of. Users can rate both movies and other user's libraries! Movie data is taken directly from IMDB, you can find any movie in the database by its title.
